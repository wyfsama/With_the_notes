### 项目介绍

一个基于 `Vue.js` `Echarts` 及 `Ant Design Vue` 的本地账本移动端项目

### 使用方法

1. `git clone`
2. `npm i`
3. `npm run serve`

### 技术栈

1. `Vue.js`
2. `Ant Design Vue`
3. `localStorage`
4. `Vue-router`
5. `Vuex`
6. `Echarts`
